# Learnings

## 2026-05-28: Vitest Test Infrastructure Setup

- **Goal**: Set up Vitest test infrastructure for the frontend project (Vue 3.2.45 + Vite 3.2.3)
- **Deps installed**: vitest@0.34.6, @vue/test-utils@2.4.6, jsdom@24.1.0
- **Vue Plugin**: vitest requires `@vitejs/plugin-vue` added to vitest.config.ts plugins array to parse `.vue` files — without it, `Failed to parse source for import analysis` error occurs
- **Lockfile**: pnpm 11.x treated the existing pnpm-lock.yaml as incompatible (generated by older pnpm). Had to delete lockfile and reinstall via `--registry https://registry.npmmirror.com` (faster network mirror)
- **Build fix**: Fresh pnpm install missed `lodash-es` (transitive dep) — had to add it explicitly for build:prod to succeed
- **Resolve aliases in vitest.config.ts**: Matched vite config `@/` → `src/`, `~/` → project root, `vue-i18n` → cjs path
- **File structure**: `src/__tests__/setup.ts`, `src/__tests__/smoke.spec.ts`, `src/__tests__/fixtures/HelloWorld.vue`
- **Verification**: `npx vitest run` → 1 passed, `pnpm run build:prod` → 2719 modules transformed
- **Scripts added**: `"test": "vitest run"`, `"test:watch": "vitest"`
